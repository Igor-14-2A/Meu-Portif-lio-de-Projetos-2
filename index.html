Jogo Pong - Scratch

Jogo Pong Duplo! Com duas raquetes!
- Leia atentamente até o FINAL, para entender como jogar!
* O jogo irá iniciar instantaneamente após clicar na bandeira verde!
* Como jogar: use o mouse para cima e para baixo para mexer as raquetes!...
* Pontos são ganhos quando acerta a raquete, ou deixar a bolha passar para a parede vermelha!
* Objetivo: Não deixe a bolinha encostar nas paredes vermelhas!
* Pare o jogo clicando no botão vermelho!
* Tente ganhar o máximo possível de postos sem perder nenhum!
